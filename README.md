A simple yet powerful web application built with **Python (Flask)** and **JavaScript (jQuery)**. This tool allows users to analyze text in real-time, including word and character counts.

- **Real-time Analysis:** Get word and character counts instantly.
- **Text Transformation:** Convert text to Uppercase or Lowercase with one click.
- **Clear Canvas:** Quickly reset the analyzer for new text.
- **Responsive Design:** Styled with Bootstrap for a clean look on all devices.


- **Backend:** Python (Flask)
- **Frontend:** HTML5, Bootstrap 5, jQuery
- **Logic:** AJAX for seamless communication between frontend and backend.
